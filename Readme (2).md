# Dry Bean Classification Using Machine Learning

## Objective
The objective of this project is to classify different types of dry beans based on their physical features using machine learning algorithms.

## Dataset Used
- Name: Dry Bean Dataset
- Source: UCI Machine Learning Repository
- Link: https://archive.ics.uci.edu/dataset/602/dry+bean+dataset
## Models Used
- Logistic Regression – simple baseline model
- K-Nearest Neighbors – distance-based classification
- Decision Tree – interpretable model
- Random Forest – high accuracy and handles complex data
- Support Vector Machine – effective in high-dimensional space

## Key Results
- Random Forest achieved the highest accuracy
- Accuracy: (91.847%)
- Model performed well in classifying different bean types

## Conclusion
Machine learning models can effectively classify dry beans. Random Forest performed best due to its ability to handle complex patterns and reduce overfitting.
