\# Dry Bean Classification using Machine Learning \& Flask



\## 📌 Objective



To develop a machine learning model to classify dry beans into different categories and deploy it using Flask.



\## 📊 Dataset



\* Name: Dry Bean Dataset

\* Source: UCI Machine Learning Repository

\* Features: 16 numerical features such as Area, Perimeter, Shape Factors



\## 🤖 Models Used



\* Random Forest (Best model)

\* Logistic Regression

\* KNN

\* SVM



\## 📈 Results



\* Accuracy: \~95% (example)

\* Best Model: Random Forest

\* Output: Predicts bean type (SEKER, BARBUNYA, HOROZ, etc.)



\## 🌐 Flask App



\* Takes user input (16 features)

\* Predicts bean type

\* Displays result in browser



\## 📂 Files Included



\* app.py

\* dry\_bean\_model.pkl

\* index.html

\* dataset.csv



\## 🚀 How to Run



1\. Install requirements:

&#x20;  pip install flask numpy joblib scikit-learn

2\. Run app:

&#x20;  python app.py

3\. Open browser:

&#x20;  http://127.0.0.1:5000/
